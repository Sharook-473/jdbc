Introduction to Bootstrap

    Bootstrap is a popular front-end framework used for
    designing responsive and visually appealing websites. 
    It is built with HTML, CSS, JavaScript and provides
    a collection of pre-designed components, utilities, 
    and responsive grid systems to simplify web development.

Key Features of Bootstrap

  1. Responsive Grid System – Uses a flexible 12-column layout to create mobile-friendly designs.
  2. Pre-designed Components – Includes buttons, forms, modals, cards, carousels, and more.
  3. CSS Utilities – Provides ready-to-use styles for spacing, colors, typography, and more.
  4. JavaScript Components – Offers interactive elements like dropdowns, tooltips, and modals.
  5. Customization – Supports theming and modifications via SASS or CSS overrides.
Conclusion
    Bootstrap simplifies web development by providing a
    robust, responsive, and customizable framework. It
    helps developers create professional-looking websit-
    es with minimal effort, making it a widely used tool
    in modern web design.
