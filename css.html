Introduction to CSS
        CSS  (Cascading Style Sheets)  is a styling  language
        used to control the presentation of web pages written 
        in HTML. It allows developers to apply styles such as
        colors, fonts, layouts, and animations to enhance the
        visual appeal and user experience of a website.
Purpose of CSS
  • Separation of Content and Design: CSS separates the structure (HTML) from the styling, making web pages easier to maintain and update.
  • Consistent Styling: It ensures a uniform look across multiple web pages.
  • Responsive Design: CSS enables websites to adapt to different screen sizes and devices.
